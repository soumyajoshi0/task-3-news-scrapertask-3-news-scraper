# News Headlines Scraper - Task 3

This project is a simple web scraper built using Python.

It fetches top news headlines from BBC News using requests and BeautifulSoup and saves them into a text file.

## Features
- Fetch HTML using requests
- Parse headlines using BeautifulSoup
- Save headlines to headlines.txt

## How to Run
1. Install libraries:
   pip install requests beautifulsoup4
2. Run:
   python scraper.py

This task helped me understand web scraping and HTML parsing in Python.
